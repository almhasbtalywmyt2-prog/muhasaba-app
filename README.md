## 🧮 طريقة حساب النسبة

### المنهجية:
1. كل عبادة لها نسبة مستقلة من 100%
2. النسبة لكل عبادة = (النقاط المحققة ÷ النقاط الكلية) × 100
3. **النسبة النهائية** = متوسط نسب جميع العبادات (15 عبادة)

### مثال عملي:
```
الورد القرآني: 3/4 = 75%
الصلوات جماعة: 4/5 = 80%
تحري الخشوع: 5/7 = 71.4%
أذكار الصلوات: 4/5 = 80%
أذكار الصباح والمساء: 2/2 = 100%
صلاة التراويح: 6/8 = 75%
صلاة الضحى: 4/4 = 100%
صلاة الوتر: 1/1 = 100%
الأذكار المئوية: 3/5 = 60%
الدعاء عند الإفطار: 1/1 = 100%
القراءة من الكتاب: 2/5 = 40%
الصدقة اليومية: 1/1 = 100%
دعاء السوق: 0/1 = 0%
الجلوس إلى الشروق: 0/1 = 0%
صلة الرحم: 1/1 = 100%

النسبة النهائية = (75 + 80 + 71.4 + 80 + 100 + 75 + 100 + 100 + 60 + 100 + 40 + 100 + 0 + 0 + 100) ÷ 15
                  = 1081.4 ÷ 15
                  = 72.1%
```

## 🚀 طريقة الاستخدام

### الخطوة 1: فتح النظام
1. افتح ملف `index.html` في المتصفح
2. أو استخدم خادم محلي (مثل Live Server)

### الخطوة 2: الحصول على الرابط الشخصي
1. اذهب إلى صفحة "النموذج اليومي"
2. اختر اسمك من القائمة
3. سيظهر رابطك الشخصي تلقائياً
4. انقر على "نسخ الرابط" 📋
5. احفظ هذا الرابط في المفضلة أو ملاحظاتك

### الخطوة 3: المحاسبة اليومية
1. افتح رابطك الشخصي
2. سيكون اسمك محدداً تلقائياً
3. املأ النموذج بصدق
4. اضغط "إرسال المحاسبة" ✅
5. ستظهر لك النسبة والمستوى فوراً

### الخطوة 4: متابعة الأداء
- **لوحة التحكم**: لرؤية الترتيب اليومي
- **تفاصيل المستخدم**: لرؤية تاريخك الكامل
- **تحليل الضعف**: لمعرفة النقاط التي تحتاج تحسين
- **التقارير**: لرؤية الأداء الأسبوعي والشهري

## 💾 تخزين البيانات

- البيانات تُحفظ في **LocalStorage** في المتصفح
- لا حاجة لخادم أو قاعدة بيانات
- البيانات محفوظة حتى لو أغلقت المتصفح
- لحذف البيانات: افتح Console واكتب `localStorage.clear()`

## 🔧 التخصيص

### إضافة مستخدم جديد:
1. افتح `index.html`
2. ابحث عن `<select id="userName">`
3. أضف سطر جديد: `<option value="الاسم">الاسم</option>`
4. افتح `script.js`
5. أضف الاسم في مصفوفة `users`

### تعديل العبادات:
1. افتح `script.js`
2. عدّل كائن `activities`
3. عدّل النموذج في `index.html` بما يتناسب
